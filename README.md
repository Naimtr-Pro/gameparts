## 🧠 Sobre o Projeto

O **GameParts** é uma aplicação web que simula uma loja virtual de hardware gamer, com funcionalidades como:

- 🔍 Busca de produtos
- 🛒 Carrinho de compras
- 🎯 Filtros por categoria e preço
- 📦 Exibição de produtos em destaque
- 📩 Formulário de contato
- 📧 Inscrição em newsletter
